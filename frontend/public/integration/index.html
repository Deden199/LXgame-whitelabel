LooxGame Wallet API Integration

Integration reference for aggregator partners.

Base URL: https://api.looxgame.com/api/
Authentication: Header X-API-Key: <your-key>
Wallet Endpoints:
  • POST /api/wallet/balance
  • POST /api/wallet/debit
  • POST /api/wallet/credit
  • POST /api/wallet/rollback

POST /api/wallet/balance

Fetch player balance using IDR integer value (no decimals).

// Request
{
  "player_id": "player-10001",
  "currency": "IDR"
}

// Response
{
  "status": "success",
  "player_id": "player-10001",
  "currency": "IDR",
  "balance": 250000
}
POST

/api/wallet/debit

Deduct stake amount from player wallet.

// Request
{
  "transaction_id": "txn-debit-9001",
  "player_id": "player-10001",
  "currency": "IDR",
  "amount": 10000,
  "game_id": "slot-gates",
  "round_id": "round-78511"
}

// Response
{
  "status": "success",
  "transaction_id": "txn-debit-9001",
  "player_id": "player-10001",
  "currency": "IDR",
  "balance": 240000
}
POST

/api/wallet/credit

Credit winnings back to player wallet.

// Request
{
  "transaction_id": "txn-credit-9002",
  "player_id": "player-10001",
  "currency": "IDR",
  "amount": 18500,
  "game_id": "slot-gates",
  "round_id": "round-78511"
}

// Response
{
  "status": "success",
  "transaction_id": "txn-credit-9002",
  "player_id": "player-10001",
  "currency": "IDR",
  "balance": 258500
}
POST

/api/wallet/rollback

Revert a previous transaction if provider retries or cancels settlement.

// Request
{
  "transaction_id": "txn-rollback-9003",
  "reference_transaction_id": "txn-debit-9001",
  "player_id": "player-10001",
  "currency": "IDR",
  "amount": 10000
}

// Response
{
  "status": "success",
  "transaction_id": "txn-rollback-9003",
  "player_id": "player-10001",
  "currency": "IDR",
  "balance": 268500
}

Idempotency

Each transaction_id is idempotent. Repeated calls return original result without modifying balance.
